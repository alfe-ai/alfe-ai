# Simple Adventure Game

A simple text-based adventure game built with Node.js.

## Features
- Text-based adventure gameplay
- Multiple locations to explore
- Simple command parsing
- Win condition

## How to Play
1. Run the game with: `npm run game`
2. Enter your name when prompted
3. Use commands to navigate:
   - `north`, `south`, `east`, `west` - Move in that direction
   - `look` - Look at current location
   - `inventory` - Show your inventory
   - `help` - Show available commands
   - `quit` - Quit the game

## Game Objective
Navigate through the locations to find the treasure room and win the game!

## Commands
- `north` (or `n`) - Move north
- `south` (or `s`) - Move south
- `east` (or `e`) - Move east
- `west` (or `w`) - Move west
- `look` - Look at current location
- `inventory` (or `i`) - Show inventory
- `help` - Show help
- `quit` - Quit game
