🥔 Potato Catcher Game 🥔

Score: 0
Time: 60s
Lives: 3

How to Play:

  • Move the basket left and right using arrow keys
  • Catch falling potatoes to score points
  • Don't let potatoes fall to the ground!
  • Each potato caught = 10 points
  • Each missed potato = 1 life lost
  • Game lasts for 60 seconds
