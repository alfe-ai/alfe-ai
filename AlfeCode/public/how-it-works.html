How it works

For each ALSH.ai Code run:

  1. A temporary working copy of your Git repository is created on the server.
  2. A new branch is created for the task.
  3. The AI coding agent executes against your prompt to implement the requested updates.
  4. The resulting changes are committed and pushed to your Git remote.

After the run is complete, you can review the diff and choose whether to merge the changes into the parent branch.
