# Potato Catcher Game

A simple HTML/JavaScript game for the Alfe AI platform where players catch falling potatoes with a basket.

## How to Play
- Use the left and right arrow keys to move the basket
- Catch falling potatoes to score points
- Avoid letting potatoes fall to the ground
- Each potato caught = 10 points
- Each missed potato = 1 life lost
- Game lasts for 60 seconds

## Features
- Simple and colorful potato-themed design
- Real-time score tracking
- Countdown timer
- Lives system
- Responsive controls

## Technical Implementation
- Pure HTML, CSS, and JavaScript (no external libraries)
- Uses requestAnimationFrame for smooth animation
- Responsive design that works on various screen sizes
- Game state management (start, pause, reset)
