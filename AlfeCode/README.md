# Number Guessing Game

A simple and fun number guessing game where you try to guess a random number between 1 and 100.

## How to Play
1. Run the game using Python 3:
   ```
   python3 number_guessing_game.py
   ```

2. The computer will generate a random number between 1 and 100
3. Try to guess the number in as few attempts as possible
4. After each guess, you'll get feedback on whether your guess was too high or too low

## Features
- Simple and intuitive interface
- Attempt counter
- Performance rating based on how many guesses you needed
- Input validation to handle invalid entries

## Requirements
- Python 3.x
