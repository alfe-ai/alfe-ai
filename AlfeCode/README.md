# Snake Game

A simple terminal-based Snake game implemented in Node.js.

## How to Play
1. Run the game with: `./run.sh snake`
2. Use WASD keys to control the snake direction:
   - W: Move up
   - A: Move left
   - S: Move down
   - D: Move right
3. Eat the 'X' food to grow and increase your score
4. Avoid hitting the walls or the snake's own body
5. Press 'Q' to quit

## Game Features
- Score tracking
- Increasing difficulty (snake speeds up as you eat more food)
- Collision detection (walls and self)
- Visual representation of snake (head as 'O', body as 'o')

## Requirements
- Node.js (version 14 or higher)
