# Porcupine Game

A fun terminal-based game where you control a porcupine to collect apples while avoiding spikes.

## How to Play

1. Run the game with: `./porcupine_game.sh`
2. Use the following keys to move:
   - W: Move up
   - A: Move left
   - S: Move down
   - D: Move right
   - Q: Quit game

## Objective

- Collect as many apples (A) as possible
- Avoid the spikes (S) - they end the game!
- Each apple gives 10 points
- Every 50 points, you advance to a new level
- The game speeds up as you progress

## Requirements

- A Unix-like terminal with bash support
- The game will not work properly in Windows cmd.exe
