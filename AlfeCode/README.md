# AlfeCode Project

This directory contains various tools and utilities for the Alfe AI project.

## Available Tools

- `run_snake.sh` - Run the snake game (requires Python 3 with curses support)
- `snake_game.py` - The snake game implementation

## How to run the snake game:

1. Make sure you have Python 3 installed
2. Run: `python3 snake_game.py`

Use arrow keys to control the snake. The goal is to eat as much food as possible without hitting the walls or yourself.
