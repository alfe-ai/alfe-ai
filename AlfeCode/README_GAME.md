# Number Guessing Game

A simple, fun guessing game where you try to guess a randomly generated number between 1 and 100.

## How to Play
1. Run the game using: `python3 guessing_game.py`
2. Enter your guess when prompted
3. Try to guess the number in as few attempts as possible!
4. You have 7 attempts to guess the number

## Features
- Random number generation between 1-100
- Attempt counter
- Feedback on whether guess is too high or too low
- Performance-based feedback messages
- Input validation

## Requirements
- Python 3.6 or higher
