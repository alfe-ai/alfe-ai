# Test Games

This directory contains test games for demonstration and testing purposes.

Currently includes:
- Number Guessing Game - Simple command-line number guessing game

## How to Run the Game

To run the number guessing game:
1. Navigate to the games directory: `cd games`
2. Run the game: `node play_game.js`

## Game Features

- Random number between 1-100
- 10 attempts to guess the number
- Feedback on whether your guess is too high or too low
- Win/lose detection
