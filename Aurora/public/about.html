About Alfe AI

Nexum is the experimental interface for the Alfe AI project. It combines chat, code generation and task management features to assist developers.

Alfe AI has a privacy-first philosophy which is why the Alfe AI code is open source on GitHub https://github.com/alfe-ai/alfe-ai-Aurelix.
