# Hangman Game

A simple hangman game implemented in bash.

## How to Run

Make the script executable and run it:

```bash
chmod +x hangman.sh
./hangman.sh
```

## Game Rules

1. Guess letters one at a time
2. You have 6 wrong guesses before the game ends
3. Try to guess the hidden word before running out of guesses
4. Enter a single letter when prompted

## Features

- Visual hangman display that updates with each wrong guess
- Tracks guessed letters
- Shows the progress of the word being guessed
- Win/loss detection
