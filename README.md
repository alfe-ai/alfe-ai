# Alfe AI Project

This repository contains the Alfe AI project. It includes various components and tools for development and deployment.

## Features
- Main application components
- Development and deployment scripts
- Documentation and configuration files

## Getting Started
1. Clone the repository
2. Install dependencies
3. Run the application using the provided scripts

## Snake Game
- Location: `AlfeCode/snake_game/`
- To run the game, use the `run_snake.sh` script:
  ```bash
  ./AlfeCode/snake_game/run_snake.sh
  ```
  Ensure you have pygame installed (`pip install pygame`).
