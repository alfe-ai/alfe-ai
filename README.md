# Guess the Number Game

A simple and fun number guessing game where you try to guess a secret number between 1 and 100.

## How to Play
1. Run the game using Python 3: `python3 guess_game.py`
2. Enter your guess when prompted
3. Try to guess the secret number in as few attempts as possible!

## Features
- Simple and intuitive gameplay
- Immediate feedback on guesses
- Tracks your number of attempts
- Input validation to ensure only numbers are entered

## Requirements
- Python 3.11.2 or higher
