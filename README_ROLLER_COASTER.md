# Roller Coaster Game

A simple HTML5/JavaScript roller coaster game created for the Alfe AI project.

## Features
- Interactive roller coaster train
- Game physics with acceleration and deceleration
- Speed and height statistics
- Start/pause and reset functionality

## How to Play
1. Click "Start Game" to begin the ride
2. Watch the roller coaster train navigate the track
3. Use "Pause Game" to pause the action
4. Click "Reset" to return to the starting position

## Technical Details
- Built with vanilla JavaScript, HTML, and CSS
- Physics simulation for train movement
- Responsive design that works on different screen sizes
- Uses CSS animations and transitions for visual effects

## File Structure
- `public/roller-coaster-game.html` - Main game page
- `public/JS/roller-coaster-game.js` - Game logic implementation

The game is accessible at `/roller-coaster-game` when the server is running.
